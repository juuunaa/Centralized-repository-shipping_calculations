# Code of Conduct

## Our Pledge
We pledge to make participation in our project a harassment-free experience for everyone.

## Our Standards
Examples of behavior that contributes to a positive environment include:
- Using welcoming language
- Being respectful of differing viewpoints

## Enforcement Responsibilities
Project maintainers are responsible for clarifying standards and enforcing them.

## Scope
This Code of Conduct applies within project spaces and public spaces when representing the project.

## Enforcement
Instances of abusive behavior may be reported to the project team.

## Enforcement Guidelines
Consequences for violations may include warnings or bans.

## Attribution
This Code of Conduct is adapted from the Contributor Covenant.
